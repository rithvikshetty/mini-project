Welcome To Student Engagement Analysis Model

This is prototype model for the Mini Project 2021. We have built a model for analysis of student engagement level in classroom.

For analysis purpose you will have to upload the csv file of facial expressions of a student generated by the 'OpenFace' software and that will be used as input for our model to predict the engagement level in classrooms.

Here, we have currently built a product-based model for the project but the main aim is to convert it to a research based study and improve the efficiency of the model.
